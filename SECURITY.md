# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in tinybpf, please report it responsibly.

**Do not open a public issue.**

Instead, use one of the following:

- **GitHub Security Advisory**: [Open a private security advisory](https://github.com/kyleseneker/tinybpf/security/advisories/new)
- **Email**: Reach out to the maintainer directly via their GitHub profile

You should receive an acknowledgment within 72 hours. A fix will be developed privately and released as soon as practical.

## Supported Versions

Only the latest release is actively supported with security fixes.
